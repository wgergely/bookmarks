bookmarks.actions.add_bookmark(server, job, root)[source]#

Save the given bookmark in user_settings.

Each bookmark is stored as dictionary entry:

bookmarks = {
    '//server/jobs/MyFirstJob/data/shots': {
        {
            common.ServerKey: '//server/jobs',
            common.JobKey:  'MyFirstJob',
            common.RootKey:  'data/shots'
        }
    },
    '//server/jobs/MySecondJob/data/shots': {
        {
            common.ServerKey: '//server/jobs',
            common.JobKey:  'MySecondJob',
            common.RootKey:  'data/shots'
        }
    }
}

Saved bookmarks can be retrieved using common.settings.get_bookmarks

Parameters:

  • server (str) – A path segment.

  • job (str) – A path segment.

  • root (str) – A path segment.

bookmarks.actions.copy_path(path, mode=0, first=True, copy=True)[source]#

Copy a file path to the clipboard.

The path will be conformed to the given mode (e.g. forward slashes converted to back-slashes for WindowsPath).

Parameters:

  • path (str) – Description of parameter path.

  • mode (int) – Any of WindowsPath, UnixPath, SlackPath or MacOSPath. Defaults to WindowsPath.

  • first (bool) – If True copy the first item of a sequence.

  • copy (bool) – If copy is false the converted path won’t be copied to the clipboard. Defaults to True.

Returns:

The converted path.

Return type:

str

bookmarks.actions.extract_zip_template(source, destination, name)[source]#

Expands the selected source zip archive to destination as name.

The contents will be expanded to a {destination}/{name} where name is an arbitrary name of a job or an asset item to be created.

Parameters:

  • destination (str) – Path to a folder

  • name (str) – Name of the root folder where the archive contents will be expanded to.

Returns:

str
