database¶

Defines BookmarkDB, the sqlite3 database interface used to store item properties, such as custom descriptions, flags, width, height, bookmarks.asset_config.asset_config values, etc.

Each bookmark item has its own database file and are stored in common.bookmark_cache_dir, in the root of the bookmark item. E.g.:

path = f'//server/job/root/{common.bookmark_cache_dir}/{common.bookmark_database}'

The database table layout is defined in TABLES, which defines the sqlite and python types for each column.

Don’t create DatabaseDB instances directly, instead use get_db() to get cached, thread-specific database controllers. E.g.:

from bookmarks import database
db = database.get_db(server, job, root)
with db.connection():
    v = db.value(*args)

bookmarks.database.get_db(server, job, root, force=False)[source]¶

Creates a database controller associated with a bookmark item.

SQLite cannot share the same connection instance between threads, hence we create and cache controllers per thread. The cached entries are stored in common.db_connections.

Parameters

  • server (str) – The name of the server.

  • job (str) – The name of the job.

  • root (str) – The name of the root.

  • force (bool) – Force retry connecting to the database.

Returns

Database controller instance.

Return type

BookmarkDB

Raises

  • RuntimeError – When the database is locked.

  • OSError – When the database is missing.

connect_with_retries()[source]¶

Connects to the database file.

The database can be locked for a brief period of time whilst it is being used by another other controller instance in another thread. This might raise an exception, but it is safe to wait on a little and re-try before deeming the operation unsuccessful.

When a database is unreachable, we’ll create an in-memory database instead, and mark the instance invalid (BookmarkDB.is_valid() will return False).

setValue(source, key, value, table='AssetData')[source]¶

Set a value in the database.

Example:

db = database.get_db(server, job, root)
with db.connection():
    source = f'//{server}/{job}/{root}/sh0010/scenes/my_scene.ma'
    db.setValue(source, 'description', 'hello world')
Parameters

  • source (str) – Source file path.

  • key (str) – A database column name.

  • value (object) – The value to set in the database.

  • table (str) – A database table.
