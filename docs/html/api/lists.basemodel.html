bookmarks.lists.basemodelÂ¶

Defines the base model used to display bookmark, asset and files items.

The model loads and reset data via the BaseModel.init_data() and BaseModel.reset_data() methods.

The model itself does not store data, but all retrieved data is cached to bookmarks.common.item_data. The interface for getting and setting data can be found in the bookmarks.common.data. However, the model is responsible for populating the data cache. See BaseModel.item_iterator().

The model exposes different datasets to the view using the BaseModel.task() and BaseModel.data_type() switches. This is because file items are stored as sequence and individual items simultaneously and because the file model also keeps separate data for each task folder it encounters.

The currently exposed data can be retrieved by BaseModel.model_data(). To change/set the data set emit the BaseModel.taskFolderChanged and BaseModel.dataTypeChanged signals with their appropriate arguments.

Actually, Bookmarks loads data in two passes. The model is responsible for discovering items, but will not populate the items with all the data, and instead we offload that onto threads. The modelâs associated threads are defined by overriding BaseModel.queues.

The base model implements data sorting via BaseModel.sort_data() but the filtering is done using FilterProxyModel.

class bookmarks.lists.basemodel.FilterProxyModel(parent=None)[source]Â¶

Proxy model responsible for filtering data for the view.

We can filter items based on the data contained in the QtCore.Qt.StatusTipRole, common.DescriptionRole and common.FileDetailsRole roles. Furthermore, based on flag values (MarkedAsArchived, MarkedAsActive, MarkedAsFavourite are implemented.)

Because of perfomarnce snags, sorting function are not implemented in the proxy model, rather in the source BaseModel.

Signals:

filterFlagChanged (QtCore.Signal): The signal emitted when the user changes a filter view setting filterTextChanged (QtCore.Signal): The signal emitted when the user changes the filter text.
