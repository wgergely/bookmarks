Python Modules#

Bookmarks works by parsing project folders to find bookmark, asset and file items. Check out the bookmark_items, asset_items and file_items for an overview of the item definitions.

The application resources are stored in the rsc folder. Basic properties can be customized by modifying the rsc/config.json file.

The application stylesheet is defined by rsc/stylesheet.json.

The bookmarks.common module contains components that define core functionality.

The list item models and views are defined in the bookmarks.items module.
